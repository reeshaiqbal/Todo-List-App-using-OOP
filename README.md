Todo List App (Python, OOP)

Overview:<br>
This is a console based Todo List application implemented in Python using Object-Oriented Programming (OOP) concepts.<br>
Features:
- Add Task: Add a new task to the list.<br>
- View Tasks: View all tasks with their current status (Pending/Done).<br>
- Mark Task as Done: Update the status of a task to “Done”.<br>
- Delete Task: Remove a task from the list.<br>
- Search Task: Search tasks by keyword.<br>
- Update Task: Modify the name of an existing task.<br>

It demonstrates the use of OOP for clean, modular, and maintainable code.

OOP Concepts in Todo List App:<br>
- Class
- Object
- Constructor (__init__) 
- Instance Variable
- Methods 
- Encapsulation
